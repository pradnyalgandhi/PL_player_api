What is this?

An API which provides Premier League Player's statistics.

A free of cost REST API which returns live JSON response of player statistics. For all the football fanatics out there, this is a perfect medium to gather statistical data of all the current Premier League players.

All data has been scraped from the Premier League website.

  • The API does not plan to do any illegal web sraping.
  • It is a free of cost RESTful API which can be called by anyone.
  • Data is refreshed after every gameweek and requires some downtime for the API to refresh the data
